## 三份报告核心结论

### 觅商机（商业情报）
- 赛道红海但有结构性空隙：中英双语一体化、攻略+工具一体化、AI驱动、新角色时效性
- 原神下行趋势明确（移动端收入3年跌83%），但仍有存量玩家
- **$5/月订阅是最大风险**：全行业无人跑通，竞品全部免费
- 建议：改为"一次性付费+捐赠"或用"AI Build Copilot"不可替代功能撑订阅
- **修正**：实际可触达市场约 42 万（海外主动搜索 build guide 的玩家），不是 210-315 万

### 柯知匠（技术评估）
- **不迁Next.js**：现有静态HTML原型质量扎实，迁移=推倒重来2-3周
- 内容Pipeline：Python + DeepSeek V3.2 + genshin.jmp.blue API
- 修正v3方案两个硬伤："DeepSeek V4 Flash"不存在→用V3.2；HoneyImpact无公开API→改用genshin.jmp.blue
- Pipeline成本极低：$0.004/篇攻略，月度$1-2
- 技术选型：Cloudflare Pages + Lemon Squeezy + Resend + Supabase(渐进式)
- **修正**：Pipeline 开发工时从 8h 上调至 20-30h；calculator.js 需从查表工具升级为真实 DMG 模拟器（技术难度被低估）

### 钱进达（运营变现）
- Reddit打法重写：5篇帖子启动=找死，必须2个月养号(karma 500+)再推广，遵守90/10规则
- v3方案遗漏HoYoLAB和Discord两个核心渠道
- "15天30篇"不现实→改为12周30篇+80个程序化页面
- 推荐Freemium模式：免费引流 + Pro增值($5/月或$40/年)
- **修正**：12个月收入从 $300-500 下调至 $40-100，$900+ 推迟到 24 个月以后
- **修正**：每篇 AI 攻略需 20-30 分钟人工审核，不是全自动

## 司令裁决（关键分歧处理）

1. **商业模式**：采纳Freemium模式，订阅生死线押注AI Build Copilot。**但须先完成 HoYoverse 法律合规检查（Phase 0）**
2. **收入预期**：12个月 $40-100/月，$300-600 推迟到 18-24 个月（原版 18-24 月 $900+ 偏乐观）
3. **内容节奏**：Pipeline批量生成 + 人工审核（每篇20-30min）+ 12周逐步发布

## 法律合规摘要（已完成调研）

HoYoverse 所有法律行动均针对泄露内容、外挂作弊和盗版分发。**零起针对合法粉丝攻略站通过订阅收费。** Fan Content Policy 覆盖实体周边，不覆盖数字服务。已有竞品（Genshin Mastery Hub）在跑 $20/月付费订阅。

**三条铁律**：①只用正式服公开数据 ②攻略永远免费，Pro 只锁工具 ③明确标注非官方粉丝站。详见 `dev-handoff.md` Phase 0.1。
