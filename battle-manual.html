原神 Build 攻略 + AI 数据工具站 — 从原型到变现全链路方案

汇编：何执策 情报：觅商机 技术：柯知匠 运营：钱进达 日期：2026-06-18
司令裁决

做，但换一种打法

三路情报汇齐后，核心判断：赛道有肉，但 v3 方案的 $5/月纯订阅制是最大风险——全行业无人跑通，竞品全部免费。调整方向：Freemium 模式（免费引流 + Pro 增值），核心护城河押注 AI Build Copilot（自然语言配队推荐），而非纯内容。时间预期从 12 个月拉长到 18-24 个月持久战。12 个月目标 $300-500/月，$900+ 推迟到 18-24 个月。

1战略概要
做什么

基于现有静态 HTML 原型，搭建原神角色 Build 攻略站 + AI 驱动的伤害计算/配队工具站。中英双语，面向全球原神玩家社区。免费攻略引流，Pro 工具变现。

为什么做

原神仍有 2100 万 MAU，10-15% 会主动搜索攻略（210-315 万人）。赛道是红海但有结构性空隙：攻略站没工具，工具站没攻略，没人做 AI 实时推荐，没人做好中英双语。一人公司用 AI 工具可以低成本切入。

现实预期：原神移动端收入 3 年跌 83%（$19亿→$3.2亿），竞品 genshin.gg 3 个月流量跌 46%。这是存量收缩市场，不是增量市场。窗口期在于"玩家更需要高效工具来优化有限投入"，而非"更多内容"。
2关键分歧裁决

三份报告在三个核心问题上出现分歧，司令裁决如下：

分歧一：$5/月订阅制能否跑通？

猎手认为大概率跑不通（竞品全免费，用户预期免费）；工匠默认走订阅设计了支付系统；操盘手认为价格 OK 但需要明显超越 Genshin Optimizer。

裁决：采纳 Freemium 模式。免费版（攻略 + 基础计算器 + 每日 3 次模拟）引流，Pro 版（$5/月或 $40/年）提供无限模拟 + 圣遗物批量优化 + 配队对比 + AI Build Copilot + 无广告。订阅制的生死线在于 AI Build Copilot 是否真的不可替代——如果它只是"另一个计算器"，用户不会付费；如果它能做到"导入 box 截图 → AI 推荐最优配队"，这就是免费工具做不到的。

分歧二：12 个月能否达到 $900/月？

操盘手修正后判断 12 个月 $185-530，$900 概率 15-20%；猎手的保守估算更低。

裁决：采纳操盘手的修正预期。12 个月目标 $300-500/月，$900+ 推迟到 18-24 个月。原因：消费级游戏 SaaS 月 churn 8-15%，免费竞品压制付费意愿，SEO 新站 6-12 个月才见效。把目标定低不是认输，是让自己能撑到第 18 个月——82% 的 micro-SaaS 死于没撑住。

分歧三：内容生产节奏 15 天 30 篇？

操盘手判断不现实（2篇/天 = 人先崩了）；工匠设计的 Pipeline 支持批量生成。

裁决：Pipeline 批量生成 + 12 周节奏发布。用 DeepSeek V3.2 Pipeline 一次性生成 30 篇攻略（技术上 4 小时可完成），但不一次性发布——按搜索热度排序，12 周内逐步发布（前 2 周发 5 篇 P0 角色，之后每周 2-3 篇）。Pipeline 解决"生产"问题，发布节奏解决"SEO + 人效"问题。

AI Build Copilot — 核心护城河

这是决定项目成败的唯一功能。如果它只是"另一个计算器"，项目失败；如果它能做到以下体验，项目有戏：

步骤用户操作系统响应
1导入游戏角色截图 / 手动选择 boxAI 识别角色 + 圣遗物
2自然语言提问："我的雷电将军怎么配队伤害最高？"AI 分析 box，排除不兼容角色
3—给出 Top 3 配队方案 + 伤害对比 + 理由
4追问："如果我把圣遗物换成如雷套呢？"实时重新模拟，给出差异

这是 Genshin Optimizer（太硬核、需手动输入）和 genshin.gg（太浅、无工具）之间的空白。一人公司用 AI 可以实际做出来。

5技术路径
核心决策：保留静态 HTML，不迁移 Next.js。现有原型（14 页面 + 988 行 CSS 设计系统 + i18n 引擎 + 计算器逻辑）质量扎实，迁移 = 推倒重来 2-3 周，对验证商业模式零帮助。等 Pro 功能需要 3+ 复杂交互组件、月收入稳定 $500+ 时再考虑迁移。

7时间线与投入
Week 1（Phase 1 启动）

上线 + Pipeline + 首批攻略

部署 Cloudflare Pages + 域名 → sitemap.xml + Google Search Console → 搭建 DeepSeek Pipeline → 生成 5 篇 P0 角色攻略 → 接入 genshin.jmp.blue 数据同步 → Google Analytics

Week 2-4

内容基建 + 社区渗透

每周 3 篇攻略 → Reddit 养号（0 推广）→ HoYoLAB 首发 → YouTube Shorts 启动 → Discord 渗透

Week 5-8（Phase 2）

支付 + Pro 功能上线

Lemon Squeezy 支付接入 → JWT + LS 订阅状态 → Pro DMG 模拟器 → 邮箱订阅 → 程序化 SEO 页面 80 个 → Reddit 首条原创内容帖

Week 9-12

变现验证

Reddit "免费工具分享"帖（大规模引流）→ Pro 转化数据分析 → AdSense 申请 → 12 周全面复盘

Month 3-6

SEO 初见效 + 口碑积累

长尾词进 Top 20-50 → 10-15 个 Pro 用户 → AdSense 批准 → 持续内容 + 社区运营

Month 6-12

增长爬坡

SEO 长尾词进 Top 10 → 25-50 个 Pro 用户（考虑 churn 后净增）→ AI Build Copilot 上线验证

Month 12-18

验证成败的窗口

如果 AI Build Copilot 验证成功 → 加速投入；如果 Pro 转化率 <1% → 转向"一次性付费 + 捐赠"模式或止损

8下一步行动
掌门现在就能做的 3 件事
  • 1
    部署上线（2h）：把现有原型推到 Cloudflare Pages，绑域名，提交 Google Search Console。先让站点可被访问、可被索引。
  • 2
    搭建 Pipeline（8h）：写 generate.py，用 DeepSeek V3.2 生成 Furina/Arlecchino/Neuvillette/Raiden/Bennett 5 篇攻略。这是验证"AI 能否批量生成质量过关攻略"的关键。
  • 3
    Reddit 养号启动（每天 30min）：注册 Reddit 账号，订阅 r/Genshin_Impact + r/GenshinImpactTips + 3 个角色主 Sub，开始浏览和回答 build 问题。2 个月后才推广，现在就种树。

  • ✕
    不要迁移 Next.js（现在）— 推倒重来对验证商业模式零帮助

成败判定线：Month 12 是关键检查点。如果 AI Build Copilot 上线后 Pro 转化率 >2%，加速投入；如果 <1%，转向"一次性付费 $15 + 捐赠"模式或考虑止损扩展到 HSR/ZZZ 多游戏。82% 的 micro-SaaS 死于没撑住——能坚持到 18 个月的人，成功率翻倍。
