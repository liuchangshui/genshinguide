纳西妲，草之神，是原神中最具影响力的草元素角色。她的元素战技（所闻遍计）可标记最多8名敌人，并在任何标记敌人受到反应伤害时造成协同草元素伤害。她的元素爆发为当前角色提供大量元素精通增益。她能触发超绽放、烈绽放、激化、蔓激化和妮露绽放——几乎所有草元素队伍都需要她。她主要依赖元素精通，但与其他元素精通角色不同，由于固有天赋4，她也能从暴击和草元素伤害中获益。

2. 最佳圣遗物套装

🎯 最佳：深林的记忆四件套

深林的记忆降低敌人30%草元素抗性。这是必需的——所有草元素伤害都受益，包括超绽放和烈绽放（造成草元素伤害）。在队伍中有两名草元素角色时，另一名可以携带深林的记忆，纳西妲则使用饰金之梦以获得更多个人伤害。

主词条优先级

部位主词条说明
🕐 时之沙 Elemental Mastery 几乎所有情况下选择元素精通沙漏。仅对六命站场输出玩法选择攻击力%（小众）。
🍷 空之杯 EM / Dendro DMG 元素精通杯（如果元素精通低于1000）。草元素伤害杯（如果元素精通已达1000，固有天赋4上限为1000）。
👑 理之冠 EM / CRIT Rate / CRIT DMG 元素精通头冠（最大化反应伤害）。暴击头冠（用于副属性良好的站场纳西妲）。

副词条优先级

  1. 1. 元素精通——达到1000（固有天赋4的软上限）。低于1000的每一点都很重要。
  2. 2. 暴击率/暴击伤害——固有天赋4将元素精通转化为暴击率（最高24%）和伤害加成（最高80%）。超过1000元素精通后，暴击变得更有价值。
  3. 3. 元素充能效率——最低130%。元素爆发很重要，但不如元素战技关键。
  4. 4. 攻击力%——价值低。纳西妲的伤害主要依赖元素精通。

3. 最佳配队方案

队伍1：超绽放
👥 Nahida + Xingqiu/Yelan + Kuki Shinobu/Raiden + Flex
典型的草元素队伍。纳西妲标记敌人，水元素创造种子，雷元素触发超绽放。久岐忍治疗并触发超绽放，全元素精通配置。低投入下伤害下限极高。
队伍2：激化/蔓激化
👥 Yae Miko/Fischl + Nahida + Kazuha + Zhongli/Baizhu
快速切换激化队伍。纳西妲提供元素精通共享以增强激化伤害。万叶降低雷元素抗性。高单体伤害。
队伍3：妮露绽放
👥 Nilou + Nahida + Kokomi + Flex Dendro/Hydro
仅限草元素+水元素。妮露创造立即爆炸的丰穰之核。纳西妲提供持续草元素附着以生成种子。大量范围伤害。

6. 常见错误

堆攻击力而非元素精通。纳西妲的伤害依赖元素精通。她的固有天赋4将元素精通转化为暴击率和伤害加成。攻击力对她作用很小。

将深林的记忆放在错误角色身上。在队伍中有另一名草元素角色（如艾尔海森）时，纳西妲可以使用饰金之梦，而另一名角色携带深林的记忆。但在单草队伍中，纳西妲必须携带深林的记忆。

未用E标记敌人。纳西妲的长按E有较大范围——在开始循环前标记所有敌人。未标记的敌人不会受到协同伤害。

元素充能效率过低。虽然她的元素爆发能量需求低（50），但在没有第二名草元素角色的队伍中，仍需130%+元素充能效率以保证稳定循环。
