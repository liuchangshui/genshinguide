关于 GenshinGuide

AI驱动，数据为准，保持更新。

我们的使命

每个版本更新都让旧攻略过时。大多数网站复制粘贴 wiki 数据，从不更新。我们创建 GenshinGuide 来解决这个问题：AI 生成的攻略会在新版本发布后 24 小时内自动更新。

工作原理

  1. 每日数据同步 — 流水线从公共 API 拉取最新游戏数据
  2. AI 生成攻略 — DeepSeek V4 基于最新数据生成结构化 Build
  3. 自动验证 — 二次校验数值与源数据的一致性
  4. 自动发布 — 验证通过的攻略附带版本标签和完整元数据上线
