单手剑 火 S 辅助 / 治疗

班尼特是《原神》中最强的辅助角色。他的元素爆发能基于自身基础攻击力（仅角色与武器——圣遗物不计算）提供巨额攻击力加成，同时为生命值低于70%的角色恢复生命。他几乎能融入所有队伍，充当治疗者、增伤者和火元素触发器。解锁1命后，其攻击力加成不再受生命值限制，使他自1.0版本起便稳坐辅助之王宝座。

2. 最佳圣遗物套装

🎯 最佳：4件套昔日宗室之仪

昔日宗室之仪在班尼特释放元素爆发后为全队提供+20%攻击力——与其自身攻击力加成完美叠加。没有比他更适合的辅助圣遗物套装。

主词条优先级

部位主词条说明
🕐 时之沙 ER% / HP% 元素充能效率优先，直至200%以上。满足充能需求后切换为生命值。
🍷 空之杯 HP% 始终选择生命值。班尼特的个人伤害次于其增伤和治疗。
👑 理之冠 Healing Bonus / HP% 治疗加成以实现最大治疗量（推荐）。若无治疗加成理之冠，则选生命值。

副词条优先级

  1. 1. 元素充能效率——直至200%以上。不可妥协。元素爆发必须在每次循环中可用。
  2. 2. 生命值——提高每次治疗量。班尼特治疗速度快，但更高生命值意味着更舒适。
  3. 3. 固定生命值——对治疗量的小幅提升。
  4. 4. 暴击率/暴击伤害——仅用于输出型班尼特（不推荐辅助）。

3. 最佳配队方案

队伍1：雷国队
👥 Raiden Shogun + Xiangling + Xingqiu + Bennett
经久不衰的版本答案队伍。班尼特同时为香菱和雷电将军增伤。香菱的旋火轮锁定班尼特的攻击力加成。雷电将军为全队充能。稳定的高额对单与范围伤害。
队伍2：国际队
👥 Childe + Xiangling + Kazuha + Bennett
毁灭性范围伤害队伍。班尼特为香菱增伤，枫原万叶双扩散火元素与水元素。达达利亚使香菱每次攻击都能触发蒸发。高操作上限，更高伤害上限。
队伍3：甘雨融化队
👥 Ganyu + Xiangling + Zhongli + Bennett
班尼特提供火元素附着，使甘雨的重击触发融化反应。钟离的护盾防止打断。班尼特的增伤被香菱的旋火轮锁定，确保稳定火元素覆盖。
队伍4：万能灵活位
👥 Any Main DPS + Any Sub DPS + Any Support + Bennett
班尼特几乎能融入所有队伍。攻击力倍率型输出角色受益最大。典型例子：流浪者、宵宫、娜维娅、林尼都依赖班尼特的增伤。

6. 常见错误

堆叠攻击力百分比圣遗物。班尼特的增伤仅基于基础攻击力（角色等级+武器攻击力）。圣遗物的攻击力百分比对其增伤毫无作用。只有武器基础攻击力才重要。

元素充能效率过低。班尼特在多数队伍中需要200%以上充能。低于180%时，元素爆发无法及时充能。必要时使用充能沙漏——一个始终可用的较弱增伤优于一个永远无法释放的强力增伤。

升级普通攻击。班尼特几乎从不使用普通攻击。将天赋书留给元素爆发和元素战技。

无脑解锁6命。班尼特的6命效果会使近战角色获得火元素附魔——这会破坏物理输出角色（优菈、雷泽）并覆盖部分元素附魔（刻晴、神里绫华）。仅当确认队伍需要时才激活6命。

将班尼特与生命值倍率型输出搭配。那维莱特、夜兰、芙宁娜等角色基于生命值而非攻击力。班尼特的攻击力加成对他们无效。应使用其他辅助。
