2. 最佳圣遗物套装

🎯 最佳：4件套「谐律异想断章」

为阿蕾奇诺量身定制。2件套提供18%攻击力。当生命之契变动时，4件套最多提供54%伤害加成。该套装与她同期上线，数学上是她的上限选择。

主词条优先级

部位主词条说明
🕐 时之沙 ATK% 始终选择攻击力。她不像雷电将军那样需要大量元素充能效率。生命值无用——她无法被治疗。
🍷 空之杯 Pyro DMG / ATK% 火元素伤害最优。攻击力杯是可行的过渡选择，但目标应为火伤杯。
👑 理之冠 CRIT Rate / CRIT DMG 追求1:2暴击比例。约80%暴击率，180%以上暴击伤害。暴击率尤其重要，因为她突破不加暴击。

副词条优先级

  1. 暴击率 / 暴击伤害（1:2比例，目标80/180+）
  2. ATK%
  3. 元素精通（仅限蒸发队，目标100-200）
  4. 元素充能效率（锦上添花，约120%即可舒适）

3. 最佳配队方案

队伍1：蒸发流阿蕾奇诺
👥 Arlecchino + Yelan + Kazuha + Bennett
👥 她的最高上限队伍。夜兰和/或行秋提供后台水元素以触发稳定蒸发。枫原万叶聚怪并用翠绿之影降低火元素抗性。班尼特提升攻击力——是的，即使他会治疗也无妨；他的攻击力加成远超过反协同。阿蕾奇诺通过元素爆发的自给自足已足够。
队伍2：纯火队
👥 Arlecchino + Xiangling + Kazuha + Bennett
👥 无反应，纯粹的火元素统治。香菱提供后台伤害和双火共鸣（+25%攻击力）。枫原万叶减抗并聚怪。班尼特同时提升阿蕾奇诺和香菱的攻击力。稳定易玩，对元素精通副词条依赖较低。
队伍3：超载队（夏沃蕾）
👥 Arlecchino + Fischl + Chevreuse + Bennett
👥 由夏沃蕾驱动的超载阵容。菲谢尔提供后台雷元素触发超载。夏沃蕾降低火/雷元素抗性并提供攻击力加成。班尼特补全队伍。对单伤害高，面对首领时表现出色。除班尼特外无需治疗。夏沃蕾的治疗量极少且非必需——她的增益来自被动技能，而非治疗。

6. 常见错误

❌ ❌ 携带班尼特以外的治疗角色。阿蕾奇诺的固有天赋一在战斗中阻止所有队友治疗。心海、白术、闲云等角色是无效位置。班尼特仅因其攻击力加成而使用。

❌ ❌ 过早使用元素爆发。她的Q清除所有生命之契并治疗。如果在普通攻击消耗生命之契前爆发，会损失伤害加成并浪费输出时间。

❌ ❌ 忘记吸收标记。使用E后，约5秒后必须使用重击吸收生命之契。否则，普通攻击造成物理伤害——几乎为零。

❌ ❌ 暴击率不足。阿蕾奇诺突破不加暴击（她加暴击伤害）。没有暴击率武器或头冠，很难达到70%以上暴击率。

❌ ❌ 使用护盾角色而非进攻辅助。钟离舒适但显著降低伤害上限。枫原万叶或砂糖提供更高的队伍输出。学会闪避。
