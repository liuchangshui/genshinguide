珊瑚宫心海 攻略

法器 水 S 辅助

海祇岛的现人神巫女。

⚡ TL;DR — 快速答案（30秒读懂）
💧 最佳武器
不灭月华 (专武) / 试作金珀 R5 (零氪)
🎯 最佳圣遗物
海染砗磲4件套 (治疗) / 千岩牢固4件套 (辅助)
👥 最佳配队
神里永冻 (神里绫华·心海·万叶·申鹤)
📊 关键属性
生命 35K+ · 充能 160-180% · 治疗头 · 战技 > 大招

珊瑚宫心海是五星水元素法器治疗角色，结合了持续的脱手水元素附着与强力的全队治疗。她的元素战技（化海月）每2秒提供范围水附着和治疗，元素爆发刷新战技持续时间并增强普攻。心海是永冻和绽放队伍的舒适之选——她将治疗和水元素附着合二为一，为队伍腾出更多输出位置。虽然常被低估，但她是游戏中最全面、最有价值的辅助之一。

2. 最佳圣遗物套装

🎯 最佳：海染砗磲4件套

治疗心海最佳。将治疗量转化为物理范围伤害（最高每3.5秒27K）。心海的持续治疗轻松叠满泡泡。

主词条优先级

部位主词条说明
🕐 时之沙 HP% / ER% 生命%强化治疗。充能%低于160%且不用试作金珀时选择。
🍷 空之杯 Hydro DMG / HP% 站场大招模式用水伤杯。纯治疗用生命杯。
👑 理之冠 Healing Bonus / HP% 治疗加成同时最大化治疗和海染泡泡伤害。生命%是过渡选择。

副词条优先级

  1. 1. 生命值% — 目标35K+
  2. 2. 元素充能效率 — 160-180%（用试作金珀可降低）
  3. 3. 元素精通 — 仅绽放/妮露队
  4. 4. 固定生命值

3. 最佳配队方案

神里永冻
👥 Ayaka + Kokomi + Kazuha + Shenhe
顶配永冻队。心海的化海月提供100%覆盖的范围水附着。将治疗+挂水合二为一。万叶聚怪+冰伤增益。
妮露绽放
👥 Nilou + Kokomi + Nahida + Flex Dendro
心海是妮露绽放的最佳驾驶员。她的治疗抵消丰穰之核的自伤。全精通配装最大化绽放伤害。
感电驾驶员
👥 Kokomi + Fischl + Beidou + Kazuha/Sucrose
站场心海驱动感电反应。海染套提供额外范围伤害。非常舒适安全的队伍。

5. 输出循环指南

神里永冻循环：
Shenhe E+Q (Cryo buff) → Kazuha E+Q (Cryo swirl + group) → Kokomi E (jellyfish, Hydro apply) → Ayaka dash + E + Q → Ayaka NA/CA spam → Refresh Kokomi E with Q before it expires

6. 常见错误

❌ 治疗心海堆攻击/暴击——被动给-100%暴击率。她不能暴击。只有生命和治疗加成有效。

❌ 不刷新水母——心海E持续12秒，冷却20秒。在消失前用Q刷新以实现100%覆盖。

❌ 站场队用生命杯而非水伤杯——大招期间心海普攻伤害可观，水伤杯提升明显。

❌ 忽略充能——没有试作金珀或足够充能，心海无法用大招刷新水母。
