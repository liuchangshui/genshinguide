枫原万叶是《原神》中首屈一指的风元素辅助角色。他能聚怪、通过翠绿之影四件套提供40%元素抗性削减，并根据元素精通为全队提供元素伤害加成。与砂糖分享元素精通不同，万叶直接提升元素伤害——这使他在纯色队和反应队中无可替代。零命已是顶级，二命则堪称逆天。

2. 最佳圣遗物套装

🎯 最佳：翠绿之影四件套

翠绿之影四件套在万叶扩散时削减敌人40%元素抗性。这是游戏中单件辅助圣遗物效果最强的。万叶的元素精通还通过其固有天赋二转化为扩散元素的元素伤害加成。三元素精通主属性可最大化两种效果。

主词条优先级

部位主词条说明
🕐 时之沙 Elemental Mastery 始终选择元素精通。辅助万叶无例外。
🍷 空之杯 Elemental Mastery 始终选择元素精通。风元素伤害加成杯仅适用于输出万叶（不推荐）。
👑 理之冠 Elemental Mastery 始终选择元素精通。暴击率头冠仅适用于需要触发被动效果的西风剑配装。

副词条优先级

  1. 1. 元素精通——每个副属性词条都很重要。目标为800-1000总元素精通。
  2. 2. 元素充能效率——最低160%。元素爆发提供8秒后台扩散及固有天赋二增益。
  3. 3. 攻击力%——少量个人伤害提升。优先级低。
  4. 4. 暴击率——仅适用于西风剑配装以触发被动。

3. 最佳配队方案

队伍1：国际队
👥 Childe + Xiangling + Kazuha + Bennett
天花板队伍。万叶双扩散水和火，同时提升公子和香菱的伤害。班尼特+万叶+香菱核心是游戏中最强的组合之一。操作要求高，但伤害无与伦比。
队伍2：冻结辅助队
👥 Ayaka + Kokomi + Kazuha + Shenhe
万叶聚拢冻结敌人并为神里绫华提升冰元素伤害。翠绿之影减抗+固有天赋二冰元素伤害加成。他的聚怪对于将敌人保持在绫华的元素爆发范围内至关重要。
队伍3：激化队
👥 Yae Miko + Fischl + Kazuha + Nahida/Baizhu
万叶扩散雷元素以触发翠绿之影减抗+雷元素伤害加成。纳西妲提供元素精通分享以提升激化伤害。万叶的聚怪有助于八重神子的杀生樱击中聚集的敌人。

6. 常见错误

未使用三元素精通主属性。辅助万叶使用攻击力%或风元素伤害加成会导致团队伤害大幅损失。他的增益随元素精通提升——请最大化它。

未扩散正确元素。万叶的固有天赋二增益适用于被扩散的元素。确保扩散你的主C使用的元素。扩散错误元素=零增益。

升级普通攻击。万叶的下落攻击伤害（来自元素战技）取决于元素战技等级，而非普通攻击。普通攻击等级完全浪费。

使用错误武器。祭礼剑看似诱人可双E，但相比铁蜂刺或西福斯的月光损失的元素精通不值得。万叶的技能冷却时间本就较短。
