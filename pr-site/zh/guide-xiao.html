魈是五星风元素长柄武器主C，通过元素爆发（靖妖傩舞）期间的下落攻击造成大量范围伤害。作为璃月三C之一，魈凭借独特的高机动性玩法和极高的伤害上限保持顶级输出地位。虽然他的扣血机制和充能需求需要精心搭配队伍，但他强大的下落攻击伤害和广泛的范围使他成为游戏中最强的对群输出之一，尤其是配上专属辅助珐露珊、芙宁娜和闲云后。

3. 最佳配队方案

顶配下落队（FFXX）
👥 Xiao + Faruzan C6 + Furina + Xianyun
魈的最强配队。珐露珊C6提供巨量风抗削减和风伤加成。芙宁娜通过气氛值放大所有伤害。闲云提升下落伤害并提供全队治疗。
经典三保一
👥 Xiao + Faruzan + Bennett + Zhongli
稳定核心。珐露珊提供风伤增益，班尼特提供巨量攻击加成，钟离护盾防止下落被打断并减抗。
双岩队
👥 Xiao + Albedo + Zhongli + Jean/Bennett
无满命珐露珊的替代方案。双岩共鸣提供护盾下15%伤害加成。阿贝多提供后台伤害。琴C4削减风抗。

Faruzan Q (Anemo buff) -> Furina E+Q (Fanfare + Hydro) -> Xianyun E+Q (heal + plunge buff) -> Xiao EE (skill x2 for energy) -> Q (burst) -> Plunge spam (10-12 high plunges) -> Repeat

6. 常见错误

❌ 不提前充能——开大前使用E×2，立即开大以在大招动画中吃到元素微粒。

❌ 使用攻击杯而非风伤杯——魈的大招将所有伤害转为风元素。风伤杯永远更好。

❌ 忽略碰撞伤害——下落过程中用模型撞击敌人可额外造成显著伤害。

❌ 不管理扣血——不带治疗时，魈的大招会持续扣血至危险水平。
